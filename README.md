## 项目简介 | Project Overview

这是一个现代化的"跨媒介数字展演创作"研究生课程展示网站，采用 React + Vite + Tailwind CSS 技术栈构建。

This is a modern website for the "Transmedia Digital Performance Creation" graduate course, built with React + Vite + Tailwind CSS.

## 功能特点 | Features

- 响应式设计，支持桌面和移动设备
- 现代化的视觉设计风格
- 流畅的动画效果和交互体验
- 完整的课程信息展示

## 技术栈 | Tech Stack

- **前端框架**: React 19
- **构建工具**: Vite 6
- **样式方案**: Tailwind CSS 4
- **动画库**: Framer Motion
- **路由管理**: React Router DOM 7
- **图标库**: Lucide React
- **UI组件**: Radix UI

## 快速开始 | Quick Start

### 安装依赖 | Install Dependencies

```bash
pnpm install
```

### 开发模式 | Development Mode

```bash
pnpm dev
```

访问 http://localhost:5173

### 构建生产版本 | Build for Production

```bash
pnpm build
```

### 预览生产版本 | Preview Production Build

```bash
pnpm preview
```
